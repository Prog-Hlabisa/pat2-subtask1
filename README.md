# Morse Code

## Overview

Morse code is a method of communication that represents letters, numbers, and punctuation using sequences of dots and dashes. It was widely used in telegraph systems and remains important in emergency communication and amateur radio.

## Historical Background

Morse code was developed in the 1830s and 1840s by Samuel Morse and Alfred Vail. It was created to transmit messages over long distances using electrical telegraph systems. Before modern telecommunications, Morse code provided a fast and reliable way to send information.

## How Morse Code Works

Each character in the alphabet is represented by a unique combination of dots (.) and dashes (-).

Examples:

A = .-
B = -...
C = -.-.
D = -..
E = .

The duration of a dash is three times longer than a dot. Spaces are used to separate letters and words.
